# TPMS App User Guide Plan

Create a comprehensive USER_GUIDE.md for the TPMS (Tyre Pressure Monitoring System) Android app. The guide will help users understand how to set up and use the app to monitor their vehicle's tyre pressures via Bluetooth LE sensors.

## Analysis Summary

The TPMS app is an Android application that:
- Scans for BLE advertisements from aftermarket TPMS sensors
- Displays live tyre pressure and temperature data on a dashboard
- Allows sensor pairing and configuration
- Supports background scanning with foreground service
- Provides alarm indicators for abnormal conditions

## User Guide Structure

1. **Introduction & Overview**
   - What the app does
   - Supported sensor types
   - Key features

2. **Getting Started**
   - Installation & permissions
   - Bluetooth requirements
   - First-time setup

3. **Main Interface Tour**
   - Dashboard tab explanation
   - Scan tab functionality  
   - Sensors tab management

4. **Sensor Setup Process**
   - How to discover sensors
   - Assigning sensors to tyre positions
   - Managing sensor configurations

5. **Reading the Dashboard**
   - Understanding pressure units
   - Temperature and battery indicators
   - Alarm conditions and warnings

6. **Settings & Customization**
   - Pressure unit preferences
   - Background scanning options
   - Known sensors filter

7. **Troubleshooting**
   - Common connectivity issues
   - Sensor not detected problems
   - Permission-related issues

8. **Technical Notes**
   - Supported sensor protocol
   - BLE advertisement format
   - Android version compatibility

## Key Content Points

- Step-by-step sensor pairing workflow
- Clear explanation of the three-tab interface
- Visual indicators and alarm conditions
- Permission requirements by Android version
- Background scanning behavior
- Settings customization options
